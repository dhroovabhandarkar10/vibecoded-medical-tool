# AI Symptom Checker & Map

A comprehensive React-based web application designed to help users identify potential medical conditions based on their symptoms and locate nearby medical facilities.

## Features

- **Symptom Checker:** Input your current symptoms to receive potential condition suggestions.
- **Results & Insights:** View detailed results based on the analyzed symptoms.
- **Map View:** Find nearby hospitals, clinics, and medical facilities via an interactive map.
- **Health History:** Track past symptom checks and health records.
- **SOS Button:** Quick access for emergency situations.

## Tech Stack

- **Frontend:** React, TypeScript, React Router
- **Styling:** Tailwind CSS, Framer Motion
- **Maps:** Leaflet, React Leaflet
- **Build Tool:** Vite

## Installation & Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Building for Production

To create a production build:
```bash
npm run build
```

This will generate the optimized files in the `dist` directory.
